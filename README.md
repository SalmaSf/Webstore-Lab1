# WebStore – Distributed Information Systems Project
Overview

This project is a web-based store application developed as part of the Distributed Information Systems course at KTH.
The goal of the project was to design and implement a web application where users can browse products and interact with the system through a typical online store workflow.

The application is built using C#, .NET and the MVC architecture. It demonstrates how a web application can manage user interaction, server-side logic and data handling in a structured way.

Features

The application allows users to:

Browse available products

View product details

Interact with the store through a web interface

Manage product-related data through backend logic

The system follows a standard Model–View–Controller (MVC) structure, separating application logic, presentation and data handling.

Technologies Used

C#

.NET

ASP.NET MVC

HTML / CSS

Git for version control

System Architecture

The application follows the MVC (Model–View–Controller) pattern:

Model – Handles application data and business logic

View – Manages the user interface and displays information to the user

Controller – Processes user input and connects the view with the model

This structure helps keep the application organized and easier to maintain.
